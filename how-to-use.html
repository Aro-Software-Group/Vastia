How to Use Vastia

Vastia transforms your audio into an immersive experience. This guide walks you through the basic flow.

Uploading Your Audio File

Start by uploading the audio file (e.g., MP3, WAV, FLAC) or video file (e.g., MP4, MOV) you want to transform. Click the 'Select Audio/Video File' button and choose a file from your device. The name of the selected file will appear below the button.

Choosing Transformation Options

Next, select the type of audio transformation you want. Options like '8D Audio', '16D Audio', and '32D Audio' are available. Once you choose an option, processing will begin automatically, and an indicator will show that Vastia is working.

Using the Preview Feature

Once processing is complete, you can preview the transformed audio. Use the 'Play' button to listen, and 'Pause' or 'Stop' to control playback. You can also compare it with the original audio using the HTML5 player.

Downloading Your Transformed File

If you're happy with the transformed audio, click the 'Download Transformed Audio' button to save your file. This button becomes active after processing is successfully completed.

Advanced Configuration (Optional)

For more detailed customization, click the 'Show Configuration' button to open the settings panel. Here, you can adjust various parameters like reverb type and panning depth. After making changes, click 'Apply Configuration' to reflect them.
